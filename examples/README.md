# Examples

These examples show the IDD workflow in action — from big idea through to Spec Kit handoff.

## Examples

| Example | Description | Shows |
|---------|-------------|-------|
| [Portfolio System](./portfolio-system/) | Modernise a legacy Excel workbook into a web app | Full 4-phase workflow, compliance-driven |

## How to Read These

Each example contains:
- `transcript.md` — the interaction between user and AI agent
- `.intent/` — the output files produced by each phase

The transcripts show real conversations, including corrections and redirects — not just happy-path outputs.
